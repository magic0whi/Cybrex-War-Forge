## 核外电子排布, 晶体结构, 分子结构与性质

1. 原子结构与性质
   1. 能层(n): 核外电子的能量是不同的, 按照电子的能量差异将其分成不同的能层, 并用符号 \\(K, L, M, N, O, P, Q\\) ... 表示相应的第1, 2, 3, 4, 5, 6, 7...能层.
   2. 能级: 同一能层的电子, 还可以把它们分成不同的能级. 能级符号的顺序是 \\(ns , np , nd , nf\\) ... (n 代表能层).
   如第3能层第二能级: 3p
   3. 能级轨道: s 电子的原子轨道都是球形的, 能层序数 n 越大, 原子轨道的半径越大; p 电子的原子轨道是纺锤型的, 每个 p 能级有 3 个原子轨道, 它们互相垂直, 类似于三维坐标轴.
   4. 轨道填充顺序: **1s2s2p3s3p4s3d4p5s4d5p6s4f5d6p7s5f6d7p** (能级三角形)
   5. 原子核外电子的排布遵循的三个原理
      1. 能量最低原理: 原子核外电子先占据能量低的轨道, 然后依次进入能量较高的轨道, 这样使整个原子处于最低的能量状态
      2. 泡利不相容定理: 每个原子轨道上最多只能容纳两个电子, 而且自旋方向相反.
      3. 洪特规则: 原子核外电子在能量相同的各个轨道上排布时, 电子尽可能分占不同的原子轨道(能级轨道), 且自旋状态相同, 这样整个原子的总能量最低.
         洪特规则特例: 能量相同的能级轨道在全满, 半满, 全空状态时, 体系的能量最低.
   6. 核外电子排布的几种表示方法
      1. 电子排布式: 用数字在能级符号的右上角标明该能级上排布的电子数. 如: K元素: \\(\ce{1s^2 2s^2 2p^6 3s^2 3p^6 4s^1}\\)

   7. 元素基本性质的周期性变化规律
      1. 原子半径: 同一周期元素(稀有气体元素除外)从左到右, 原子半径逐渐减小; 同一主族元素从上到下, 原子半径逐渐增大.
      2. 第一电离能: 某元素的气态原子失去一个电子形成 +1价 气态阳离子所需的最低能量.
         第一电离能越小, 电子越容易被夺走. 同一周期的主族元素从左到右, 第一电离能呈现增大趋势;
         同一主族元素从上往下, 第一电离能逐渐减小.
      3. 电负性: 同一周期的主族元素从左到右, 电负性逐渐增大; 同一主族元素从上到下, 电负性递减.
      4. 元素下金属性和非金属性: 同一周期元素(稀有气体元素除外), 从左到右, 从上到下, 元素金属性逐渐增强, 非金属性逐渐减弱.

   4. 判断晶体类型的方法
      1. 依据组成晶体的晶格质点和质点间的作用判断:
         离子晶体的晶格质点是阴阳离子, 质点间作用是离子键;
         原子晶体的晶格质点是原子, 质点间作用是共价键;
         分子晶体的晶格质点是分子, 质点间作用是分子间作用力;
         金属晶体的晶格质点是金属阳离子和自由电子, 质点间作用是金属键.
      2. 依据物质的分类判断: 
         1. 离子晶体: 活泼金属氧化物, 强碱和绝大多数的盐类;
         2. 原子晶体: 
            单质有金刚石, 晶体硅, 晶体硼等;
            化合物有碳化硅, 二氧化硅等;
         3. 分子晶体: 大多数非金属单质, 气体氢化物, 非金属氧化物, 酸, 绝大多数有机物;
         4. 金属晶体: 常温下金属单质(除汞外)是金属晶体.
      3. 依据导电性判断:
         1. 离子晶体在水溶液或融化时能导电;
         2. 原子晶体一般为非导体;
         3. 分子晶体为非导体, 而分子晶体中的电解质(主要是酸和非金属氢化物)溶于水能导电,
            这是因为分子内的化学键断裂形成了自由移动的离子, 但它不再是分子晶体了;
         4. 金属晶体是电的良导体
      4. 依据硬度和机械性能判断
         离子晶体硬度较大, 但是脆;
         原子晶体硬度大;
         分子晶体硬度小且较脆;
         金属晶体多数硬度大, 具有延展性.
      5. 晶体熔沸点高低的比较
         1. 不同晶型
            一般来说, 分子晶体 < 离子晶体 < 原子晶体
         2. 同类晶型
            1. 离子晶体: 熔沸点高低取决于离子键强度.
               一般来说, 离子半径越小, 所带电荷越多, 强度越大
               如: 离子半径: \(\ce{KBr < KCl < KF}\), 所带电荷: \\(\ce{CsCl < NaCl < MgCl2 < MgO}\\)
            2. 原子晶体: 熔沸点高低取决于共价键的键能和键长.
               键能越大, 键长越短, 熔沸点越高.
               如: 晶体硅 < 碳化硅 < 金刚石
            3. 分子晶体: 熔沸点高低取决于分子间作用力(范德华力)的大小.
               1. 一般来说, 对于组成和结构相似的由分子组成的物质,
                  质量越大 -> 范德华力越大 -> 熔沸点越高
                  如沸点: \\(\ce{N2 < O2}\\) .
               2. 但在同分异构体中, 支链多的沸点反而低.
                  如沸点: \\(\ce{新戊烷 < 异戊烷 < 正戊烷}\\)
               3. 如果分子间存在氢键, 其熔沸点会比同类高
         3. 金属晶体: 熔沸点高低取决于金属键强弱.
            一般来说: 金属离子半径越小, 所带电荷越多, 金属键越强.
            如: \\(\ce{Na < Mg < Al}\\)

      2. 价层电子对互斥理论
         分子中的价电子对(包括成键电子对和弧电子对)由于相互排斥作用, 而趋向于尽可能远离以减少斥力, 分子尽可能采取对称的空间构型.
         对于 \\(\ce{AB_m}\\) 型分子(A 是中心原子, B 是配位原子), 分子的价电子对数可以通过下式确定:
         \\(n=\dfrac{配位原子提供的价电子数 \times m}{2}\\)
         其中, 中心原子的价电子数等于中心原子的最外层电子数, 配位原子中卤素原子, 氢原子提供一个价电子, 氧原子和硫原子按不提供价电子计算.

   4. 氢键: 氢键是一种既可以存在于分子之间又可以存在于分子内部的作用力. 它比化学键弱, 比范德华力强. 当 H 原子与电负性大而原子半径小的原子 X 以共价键结合时, H 原子能够跟另一个电负性大的原子 Y 之间形成氢键.
      氢键对物质性质的影响:
      1. 氢键不是化学键, 属于分子间作用力. 除 \\(\ce{NH3 , H2O , HF}\\) 外, 同主族元素的简单氢化物的沸点还遵循: 相对分子质量增大 -> 分子间作用力增大 -> 熔沸点升高 的规律.
      2. 分子间氢键的形成除使物质的熔沸点升高外, 对物质的溶解度, 硬度等也有影响. 如 \\(\ce{NH3}\\) 极易溶于水(1 体积水能溶剂 700 体积 \\(\ce{NH3}\\)), 原因就在于氨分子和水分子之间形成了氢键, 彼此绨合, 增大了 \\(\ce{NH3}\\) 的溶解度. 再如, 冰的硬度比一般固态共价化合物的硬度大, 也是因为冰中有氢键.
      3. 互为同分异构体的物质, 熔沸点: 分子内形成氢键的物质 < 分子间形成氢键的物质
   5. 等电子原理
      1. 等电子体: 
         a. 原子数相同, 价电子总数相同的粒子(粒子包括分子和离子)互称为等电子体. 如 \\(\ce{N2}\\) 与 \\(\ce{CO}\\) 互为等电子体 (原子数相同<->它们都是两个元素组成的分子).
         b. 由短周期元素组成的粒子, 只要其原子数相同, 各原子最外层电子数之和相同, 即可互称为等电子体. 如 \\(\ce{O3}\\) 与 \\(\\ce{SO2}\) 互为等电子体.
      2. 等电子原理基本观点: 化学通式相同且价电子总数相等的分子或粒子具有相同的空间构型和化学键类型等结构特征.
      3. 常见的等电子体
         \\(\ce{N2}\\) 与 \\(\ce{CO}\\) ; \\(\ce{CO2}\\) 与 \\(\ce{N2O}\\) ; \\(\ce{O3}\\) 与 \\(\ce{NO2-}\\);
         \\(\ce{CO3^2-}\\) , \\(\ce{NO3-}\\) 与 \\(\ce{SO3}\\) ; \\(\ce{SiO4^4-}\\) , \\(\ce{PO4^3-}\\) 与 \\(\ce{ClO4-}\\) ,
         苯 与 \\(\ce{B3N3H6}\\) (硼氮苯)等.
      4. 等电子原理的应用
         1. 根据等电子原理可以较快判断一些分子的构型.
            如 \\(\ce{NH3}\\) 和 \\(\ce{H3O+}\\);
            \\(\ce{SiCl4 , SiO4^4-}\\) 和 \\(\ce{SO4^2-}\\) 的空间构型相似, 分别是三角锥形和正四面体形.
         2. 应用于新材料领域. 某些等电子体具有相同的结构特征, 表现在性质和应用上也有相同之处, 这对制造新材料是有启发的.
            列如, 晶体硅和锗是良好的半导体材料, 它们的等电子体磷化铝(AlP) , 砷化镓(GaAs)也都是良好的半导体材料.
